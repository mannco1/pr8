# Классы 3 и 4 типа

Эти классы предназначены для решения уравнений вида C = 0 и A*x^2 + C = 0, где A, C - заданные коэффициенты.

# Класс type3

Класс type3 имеет следующие поля и методы:

- Поле C - коэффициент уравнения C = 0
- Конструктор type3(double c1) - принимает значение коэффициента C и инициализирует им поле C
- Метод void Get_answer() - выводит на экран сообщение "Корней нет", так как уравнение C = 0 не имеет корней, если C != 0
- Метод void show() - выводит на экран уравнение вида C = 0

# Класс type4

Класс type4 имеет следующие поля и методы:

- Поле A - коэффициент при x^2 в уравнении A*x^2 + C = 0
- Поле C - свободный коэффициент в уравнении A*x^2 + C = 0
- Конструктор type4(double a1, double c1) - принимает значения коэффициентов A и C и инициализирует ими поля A и C
- Метод void Get_answer() - решает уравнение A*x^2 + C = 0 и выводит на экран его корни или сообщение "Корней нет", если они отсутствуют
- Метод void show() - выводит на экран уравнение вида A*x^2 + C = 0
